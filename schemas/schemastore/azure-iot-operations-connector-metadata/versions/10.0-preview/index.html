The default MQTT topic that will be published to. Values are allowed to contain deploy-time parameters like 'mqtt/{deviceName}/{inboundEndpointName}/myMessages'. Dataset/eventGroup/event/stream name tokens are only allowed if the configured destination is for a dataset/event/stream.
